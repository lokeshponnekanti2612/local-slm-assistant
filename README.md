# Local SLM Assistant

An offline AI assistant built using a small language model (SLM), focusing on local inference, streaming, and benchmarking.

## Current Features
- Ollama-based local inference
- Streaming responses
- Benchmarking latency and throughput

## Project Structure
- `app/` → core logic
- `experiments/` → runnable scripts
- `assistant/` → future orchestration
- `results/` → outputs
